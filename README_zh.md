# Reading Plugin 使用说明

这是一个面向 JetBrains IDE 的阅读插件原型。当前已实现 JetBrains 版本，VS Code 版本目录暂时只是占位。

## 如何打开插件

安装并重启 WebStorm 后，有三种入口：

1. 右侧工具窗口栏点击 `Reader`。
2. 菜单进入 `视图 > 工具窗口 > Reader`。
3. 菜单进入 `工具 > Reader`，直接执行单个命令。

如果看不到 `Reader` 工具窗口，先确认插件已启用：

`Settings / Preferences > Plugins > Installed > Reader`

## 操作面板

`Reader` 工具窗口里有这些按钮：

- `打开 TXT / EPUB`：选择本地 `.txt` 或 `.epub` 文件。
- `显示原生提示层`：在当前编辑器里显示阅读浮层。
- `隐藏原生提示层`：关闭阅读浮层。
- `打开目录`：显示章节列表，选择章节后跳转。
- `下一章`：跳到下一章。
- `自动下一章 开/关`：按指定秒数自动切换章节。
- `平台网页登录`：打开番茄、起点、微信读书或自定义网页，用于登录和查看平台内容。
- `网页正文提取`：输入第三方小说网页 URL，插件会提取正文并进入阅读模式。

## 导入 TXT / EPUB

1. 打开任意编辑器文件，保证当前有编辑器区域。
2. 打开 `Reader` 工具窗口。
3. 点击 `打开 TXT / EPUB`。
4. 选择文件。
5. 输入章节拆分正则。
6. 输入 TXT 编码，默认 `UTF-8`。

默认章节正则：

```regex
(?m)^\s*(第\s*[0-9一二三四五六七八九十百千万零〇两]+\s*[章节卷回部篇].*)$
```

正则的第一个捕获组会作为章节标题。EPUB 会优先按内部 HTML 文件拆分；TXT 会按正则拆分。

## 网页模式

### 平台网页登录

点击 `平台网页登录`，选择：

- 番茄小说
- 起点中文网
- 微信读书
- 自定义 URL

这个模式用于打开平台页面和登录。当前版本不会自动抓取登录后的书架内容，只提供内嵌浏览器入口。

### 网页正文提取

点击 `网页正文提取`，输入小说网页 URL。插件会尝试：

- 过滤脚本、样式、图片、iframe 等无关内容。
- 优先提取 `article`、`main`、`content`、`chapter`、`read`、`book` 相关正文容器。
- 将正文文本导入阅读状态。

## 快捷键

默认快捷键：

| 功能 | 默认快捷键 |
| --- | --- |
| 打开 TXT / EPUB | `Alt+Shift+O` |
| 显示原生提示层 | `Alt+Shift+R` |
| 隐藏原生提示层 | `Alt+Shift+H` |
| 打开目录 | `Alt+Shift+T` |
| 下一章 | `Alt+Shift+N` |
| 自动下一章 开/关 | `Alt+Shift+A` |
| 平台网页登录 | `Alt+Shift+B` |
| 网页正文提取 | `Alt+Shift+W` |

自定义快捷键：

1. 打开 `Settings / Preferences > Keymap`。
2. 搜索 `Reader`。
3. 找到对应动作，例如 `Open Local Book`、`Show Native Reader`。
4. 右键动作，选择 `Add Keyboard Shortcut`。

如果快捷键被系统或 IDE 占用，WebStorm 会提示冲突，可以按提示移除或换一个组合。

## 当前限制

- 当前版本是 JetBrains 插件原型，不是最终发布版。
- 平台网页登录模式目前只负责打开平台页面，不自动同步书架。
- 网页正文提取基于通用规则，复杂站点可能需要后续增加站点适配规则。
- EPUB 解析当前使用内部 HTML 文件顺序，不解析完整 OPF spine 元数据。
