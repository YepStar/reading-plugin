# Reading Plugin 使用说明

这是一个面向 JetBrains IDE 的阅读插件原型。当前已实现 JetBrains 版本，VS Code 版本日后待实现。

采用jetbrain原生显示层，让你摸鱼更大胆。

主要功能为：
1. 可直接传epub或txt文件，根据输入的章节正则自动拆分章节(可跳转/本地化记忆)。
2. 浏览器页面模式:可打开番茄/起点等平台登录页面，直接阅读平台书架内容。
3. 类夸克浏览器书架模式:打开第三方小说网页，自动过滤无关图片网页，只显示正文文本。
4. 在线书源模式:内置 W-Reader 的部分默认书源，可搜索小说、选择目录章节并导入原生提示层，同时支持书源管理。

## 如何打开插件

安装并重启 WebStorm 后，有三种入口：

1. 右侧工具窗口栏点击 `Reader-plugin-yip`。
2. 菜单进入 `视图 > 工具窗口 > Reader-plugin-yip`。
3. 菜单进入 `工具 > Reader-plugin-yip`，直接执行单个命令。

如果看不到 `Reader` 工具窗口，先确认插件已启用：

`Settings / Preferences > Plugins > Installed > Reader-plugin-yip`

## 操作面板

`Reader-plugin-yip` 工具窗口里有这些按钮：

- `打开 TXT / EPUB`：选择本地 `.txt` 或 `.epub` 文件。
- `显示/隐藏原生提示层`：用同一个动作切换阅读浮层；如果之前导入过本地书籍，会按保存的文件路径、正则、章节和滚动位置恢复。
- `打开目录`：显示章节列表，选择章节后跳转。
- `上一章`：跳到上一章；在线书源章节会自动加载正文。
- `下一章`：跳到下一章。
- `自动下一章 开/关`：按指定秒数自动切换章节。
- `平台网页登录`：打开番茄、起点、微信读书或自定义网页，用于登录和查看平台内容；关闭窗口后会保存最后访问的页面，下次可以直接打开。
- `网页正文提取`：输入第三方小说网页 URL，插件会提取正文并进入阅读模式。
- `在线书源搜索`：使用启用的书源搜索小说，选择搜索结果和章节后导入原生提示层。
- `书源管理`：新增、编辑、启用/停用、删除、重置在线书源。

## 导入 TXT / EPUB

1. 打开任意编辑器文件，保证当前有编辑器区域。
2. 打开 `Reader-plugin-yip` 工具窗口。
3. 点击 `打开 TXT / EPUB`。
4. 选择文件。
5. 输入章节拆分正则。
6. 输入 TXT 编码，默认 `UTF-8`。

默认章节正则：

```regex
(?m)^\s*(第\s*[0-9一二三四五六七八九十百千万零〇两]+\s*[章节卷回部篇].*)$
```

正则的第一个捕获组会作为章节标题。EPUB 会优先按内部 HTML 文件拆分；TXT 会按正则拆分。

导入成功后，插件会在当前项目的配置中保存：

- 本地文件路径
- 章节正则
- TXT 编码
- 当前章节
- 原生提示层滚动位置

下次打开项目后，点击 `显示/隐藏原生提示层` 即可恢复上次本地阅读位置。

## 网页模式

### 平台网页登录

点击 `平台网页登录`，选择：

- 番茄小说
- 起点中文网
- 微信读书
- 自定义 URL

这个模式用于打开平台页面和登录。关闭平台窗口时，插件会保存最后访问的页面。下次点击 `平台网页登录` 时，可以选择 `上次页面` 继续打开。

当你已经登录，并进入平台书架中的小说阅读页面后，可以点击平台窗口顶部的 `将当前页面导入原生提示层`。插件会直接从当前浏览器页面读取 DOM 内容，过滤无关内容后导入原生提示层。这个方式不依赖额外 HTTP 请求，因此可以读取登录后的页面正文。

### 网页正文提取

点击 `网页正文提取`，输入小说网页 URL。插件会尝试：

- 过滤脚本、样式、图片、iframe 等无关内容。
- 优先提取 `article`、`main`、`content`、`chapter`、`read`、`book` 相关正文容器。
- 将正文文本导入阅读状态。

## 在线书源

点击 `在线书源搜索` 后：

1. 选择启用的书源。
2. 输入书名或关键词。
3. 选择搜索结果。
4. 选择目录中的章节。
5. 插件会加载正文并显示在编辑器内原生提示层。

搜索结果和章节目录使用可滚动列表展示，适合上千章目录。

当前内置书源统一维护在仓库根目录 `book-source-config/` 下，并会随插件一起打包。当前包含：

- 笔趣阁
- 酷我小说

在线书源章节会保留完整目录、当前章节和已加载正文。使用 `上一章`、`下一章`、`自动下一章` 或 `打开目录` 跳转到未加载章节时，插件会按当前书源自动拉取该章节正文。重新打开项目后，点击 `显示/隐藏原生提示层` 会恢复上次在线阅读位置。

### 书源管理

点击 `书源管理` 可以：

- 新增书源 JSON
- 编辑当前书源 JSON
- 启用或停用书源
- 设为默认书源
- 删除书源
- 重置内置书源

内置书源配置文件位于 `book-source-config/`。项目内通过 `书源管理` 修改后的书源仍会保存到当前项目的 `reader-book-sources.xml` 中。JSON 字段包括 `search`、`catalog`、`content`、`info`、`processor` 规则。详细格式见 `book-source-config/README_zh.md`。

## 快捷键

默认快捷键：

| 功能 | 默认快捷键 |
| --- | --- |
| 打开 TXT / EPUB | `Alt+Shift+O` |
| 显示/隐藏原生提示层 | `Alt+Shift+R` |
| 打开目录 | `Alt+Shift+T` |
| 上一章 | `Alt+Shift+P` |
| 下一章 | `Alt+Shift+N` |
| 自动下一章 开/关 | `Alt+Shift+A` |
| 平台网页登录 | `Alt+Shift+B` |
| 网页正文提取 | `Alt+Shift+W` |
| 在线书源搜索 | `Alt+Shift+S` |
| 书源管理 | `Alt+Shift+M` |

自定义快捷键：

1. 打开 `Settings / Preferences > Keymap`。
2. 搜索 `Reader-plugin-yip`。
3. 找到对应动作，例如 `打开本地 TXT / EPUB`、`显示/隐藏原生提示层`。
4. 右键动作，选择 `Add Keyboard Shortcut`。

如果快捷键被系统或 IDE 占用，WebStorm 会提示冲突，可以按提示移除或换一个组合。

## 余留问题

- 当前版本是 JetBrains 插件原型，不是最终发布版。
- 平台网页登录模式会保存最后访问页面，并支持把当前阅读页导入原生提示层，但不会自动同步完整书架。
- 网页正文提取基于通用规则，复杂站点可能需要后续增加站点适配规则。
- 在线书源已支持 W-Reader 默认书源中的 HTML/JSON 规则子集和基础 processor；依赖站点自定义脚本、加密解密的源需要后续单独适配。
- EPUB 解析当前使用内部 HTML 文件顺序，不解析完整 OPF spine 元数据。

## 打包与发布

打包 JetBrains 插件：

```bash
scripts/package-jetbrains.sh
```

产物：

```text
jetbrains/build/distributions/reader-jetbrains-0.1.0.zip
```

发布到 JetBrains Marketplace 前先准备 token：

```bash
export JETBRAINS_MARKETPLACE_TOKEN=...
scripts/publish-jetbrains.sh
```

版本号固定在 `jetbrains/gradle.properties` 和 `plugin.xml`，当前为 `0.1.0`。发布脚本默认上传到 `default` 通道，可用 `PLUGIN_CHANNELS=eap` 覆盖。Token 不要提交到仓库。
